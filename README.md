# React Todo App (Vite)

A simple but complete Todo application built using **React** and **Vite**.  
The app supports adding, editing, completing, and deleting tasks, with basic validation to prevent duplicate entries.

This project was built as a hands-on learning exercise to deeply understand React fundamentals such as state management, component communication, and immutable updates.
